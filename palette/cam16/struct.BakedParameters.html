Struct BakedParameters

Source 
pub struct BakedParameters<WpParam, T> { /* private fields */ }
Expand description

Pre-calculated variables for CAM16, that only depend on the viewing conditions.

Derived from Parameters, the BakedParameters can help reducing the amount of repeated work required for converting multiple colors.
