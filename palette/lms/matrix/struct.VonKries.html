Struct palette::lms::matrix::VonKries

source · 
pub struct VonKries;
Expand description

Represents the matrix used with the von Kries transform method (MvonKries).

It’s also known as the Hunt-Pointer-Estevez matrix (MHPE) and was originally used in conjunction with the von Kries method for chromatic adaptation. It’s also used in the Hunt and RLAB color appearance models.
